Установка и настройка плагина WordPress

  1. Скачайте плагин для WordPress - Скачать

  2. Распакуйте архив и закачайте его содержимое в папку "wp-content/plugins". На сервере у вас должна получиться такая структура директорий:

  3. Зайдите на сайт под администратором, перейдите в панель управления, далее выберите раздел "Плагины" (1), и активируйте плагин "MoneyInst.com" (2)

  4. Перейдите в раздел "Настройки" (1) и выберите "MoneyInst.com" (2)

  5. Настройка плагина:

    • Ваш ид в партнерской программе MoneyInst
    • Домены для которых необходипо производить замену ссылок. Каждый новый домен необходимо вводить с новой строки без указания протокола (http://, https:// ...), напрмер: example.com. Если это поле оставить пустым, то домен в ссылке не будет анализироваться, т.е. будет производиться подмена ссылки для любого домена.
    • Расширения файлов для которых необходимо подменять ссылки. Расширения вводятся через запятую бех точки, например: exe, rar, zip, pdf. Если это поле оставить пустым, то расширение файла в ссылке не будет анализироваться, т.е. будет производиться подмена ссылки для любого расширения.
    • При замене ссылок, плагин пытается автоматически определить тип файла. Если ему это не удается, то используется тип файла по умолчанию. Рекомендуется выбирать такой тип, файлы которого вы в большинстве случаев будете предлагать.Напрмер, у вас на сайте в большинстве случаев будут вылаживать архивы, тогда нужно изменить значение этого поля на "archive".
    • После изменения настроек, нажмите кнопку "Сохранить изминения".

Установка и настройка модуля DLE

  1. Сначала необходимо определить какую версию DLE вы используете (Win1251 и UTF8). Узнать версию можно просмотрев исходный код любой страницы вашего сайта (вверху вы увидите метатег используемой кодировки).

    Скачайте соответствующий модуль:
    Win1251
    UTF8

  2. Распакуйте архив и закачайте cодержимое папку «engine» на сервер, совмещая ее содержимое с содержимым уже имеющиихся папок.

  3. Зайдите на сайт под администратором и перейдите по адресу http://ваш_сайт/admin.php?mod=moneyinst

  4. Настройка модуля:

    • Если флажок отмечен, модуль будет производить подмену ссылок, иначе - нет.
    • Ваш ид в партнерской программе MoneyInst
    • Домены для которых необходипо производить замену ссылок. Каждый новый домен необходимо вводить с новой строки без указания протокола (http://, https:// ...), напрмер: example.com. Если это поле оставить пустым, то домен в ссылке не будет анализироваться, т.е. будет производиться подмена ссылки для любого домена.
    • Расширения файлов для которых необходимо подменять ссылки. Расширения вводятся через запятую бех точки, например: exe, rar, zip, pdf. Если это поле оставить пустым, то расширение файла в ссылке не будет анализироваться, т.е. будет производиться подмена ссылки для любого расширения.
    • Тип файла по умолчанию. Иногда бывают ситуации когда из адреса ссылки невозможно определить тип файла, тогда для него будет браться значвение из этого поля. Рекомендуется выбирать такой тип, файлы которого вы в большинстве случаев будете предлагать.Напрмер, у вас на сайте в большинстве случаев будут вылаживать архивы, тогда нужно изменить значение этого поля на "archive".
    • Настройка определяет для пользователей из каких груп будет происходить замена ссылок. Например, если в этом поле не отметить «Администраторы», то для пользователей из этой группы ссылки не будут подменяться.
    • Настройка определяет какие ссылки необходимо заменять.
    • После изменения настроек, нажмите кнопку "Сохранить изминения".

  5. Перейдите в раздел «Настройка скрипта» -> «Настройка системы» (1,2). Выберите «файлы» (3), выключиите настройку «Блокировать скачивание файлов с других серверов» (4), нажмите «Сохранить» (5)

Настройка Ucoz для работы с партнерской программой

  1. Зайдите в панель управления вашим сайтом.

  2. Перейдите в раздел "Каталог файлов" (1), нажмите ссылку "Управление дизайном модуля" (2).

  3. В списке страниц выберите "Страница материала и комментариев к нему" (1), в области редактирования замените текст (2)
    <a href="$FILE_URL$"> на <a href="$FILE_URL$" download_url="$FILE_DIRECT_URL$" class="mi-download-link" download_sid="1">

    Обратите внимание, что в качестве значения атрибута download_sid необходимо ввести ваш ид в партнерсокй программе MoneyInst (3).

    Если вы хотите чтобы имя файла соответствовало названию материала добавьте в ссылку следующий атрибут: download_name="$ENTRY_TITLE$"

  4. В области редактирования перед закрывающим тегом </body> (1) вставьте этот текст После нажмите кнопку «Сохранить» (2).

Универсальный скрипт для подмены ссылок

  1. Скачайте архив. Распакуйте и загрузите на сервер файл скрипта.

  2. Добавьте ссылку на скрипт в хеадер вашего сайт. <script src="//путь_к_папке_куда_закачали_скрипт/moneyinst.js"></script>

    Ссылка на скачиваемый файл должна иметь следующий вид: <a download_url="(1)" download_sid="(2)" class="mi-download-link" ...

    • (1) - прямая ссылка на файл (при переходе по этой ссылке начинается скачивание файла). Этот параметр можно не указывать если в href и так находится прямая ссылка на файл.
    • (2) - ваш ид в партнерской программе MoneyInst
    • обратите внимание что ссылка должна содержать класс "mi-download-link"

    Для того чтобы самому задать имя скачиваемого файла добавьте параметр: download_name="имя_файла.расширение"

Универсальный скрипт для подмены ссылок (PHP + Javascript)

  1. Скачайте архив. Распакуйте и загрузите на сервер в корень сайта папку «moneyinst».

  2. Добавьте ссылку на скрипт в хеадер вашего сайт. <script src=«//moneyinst/moneyinst.js»></script>

  3. Ссылка на скачиваемый файл должна иметь следующий вид: <a download_url=«(1)» download_sid="(2)" class=«mi-download-link» ...

    • (1) - прямая ссылка на файл (при переходе по этой ссылке начинается скачивание файла). Этот параметр можно не указывать если в href и так находится прямая ссылка на файл.
    • (2) - ваш ид в партнерской программе MoneyInst
    • обратите внимание что ссылка должна содержать класс «mi-download-link»

    Для того чтобы самому задать имя скачиваемого файла добавьте параметр: download_name=«имя_файла.расширение»

Универсальный скрипт для подмены ссылок (PHP)

  1. Скачайте архив. Распакуйте, откройте в блокноте и исправьте значение переменной $sid на ваше (2-я строчка). Загрузите на сервер исправленный скрипт.

  2. Ссылка на скачиваемый файл должна иметь следующий вид: <a href=«/путь_к_папке_куда_закачали_скрипт/download.php?file=(1)» ...

    • (1) - закодированная в base64 ссылка на скачиваемый файл (при открытии этой ссылки в браузере происходит загрузка оригинального файла).

    Для того что б самому задать имя скачиваемого файла добавьте параметр: name="имя_файла.расширение" Т.е. ссылка примет следующий вид: <a href="/путь_к_папке_куда_закачали_скрипт/download.php?file=(1)&name=(2)" ...

    • (1) - закодированная в base64 ссылка на скачиваемый файл (при открытии этой ссылки в браузере происходит загрузка оригинального файла).
    • (2) - закодированное в base64 имя файла с расширением (например: archive.rar).
